# Color Palette Generator

Generate beautiful color palettes for your design projects. Extract colors from images and create harmonious color schemes.

## Features
- Generate color palettes
- Extract colors from images
- Color harmony rules
- Copy color codes
- Mobile-friendly

## Tech Stack
- HTML/CSS/JavaScript
- Cloudflare Pages
